# 小东AI助手展示面板

东品西选电商服务（成都）有限公司 - 专业跨境电商助手系统

## 🚀 功能特性

### 多线程可视化
- 10线程池实时状态监控
- 活跃/空闲线程可视化
- 任务进度实时更新

### 系统监控
- CPU/内存/磁盘使用率
- 关键进程状态监控
- 性能指标实时显示

### 任务调度
- 今日/明日/本周/本月计划
- 任务优先级管理
- 完成状态实时跟踪

### 数据统计
- 多平台数据仓库统计
- 每日任务完成情况
- 趋势分析和增长指标

### 知识库管理
- 记忆文件管理
- 已安装技能展示
- 学习进度跟踪

- **部署平台**: GitHub Pages + GitHub Actions
